# Changelog

Tất cả các thay đổi quan trọng của dự án sẽ được ghi lại trong file này.

## [1.0.0] - 2024-12-19

### Added
- ✅ Khởi tạo dự án Capstone NextJS
- ✅ Cấu hình Vite với TypeScript
- ✅ Setup Redux Toolkit cho state management
- ✅ Implement authentication (login/register)
- ✅ Tạo các trang chính: Home, Listings, RoomDetail, Profile
- ✅ Responsive design với Tailwind CSS
- ✅ API integration với Cybersoft API

### Changed
- 🔧 Cải thiện cấu hình Vite cho deployment
- 🔧 Tối ưu hóa build process với code splitting
- 🔧 Cải thiện error handling cho API calls
- 🔧 Thêm environment variables support

### Fixed
- 🐛 Khắc phục lỗi routing khi deploy
- 🐛 Cải thiện xử lý lỗi network
- 🐛 Tối ưu hóa performance cho production

### Technical Details
- **Framework**: React 19.1.0
- **Build Tool**: Vite 7.1.2
- **State Management**: Redux Toolkit 2.8.2
- **Styling**: Tailwind CSS 4.1.11
- **HTTP Client**: Axios 1.11.0
